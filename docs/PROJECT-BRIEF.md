# 辽轨智能实训能力评估平台 - 项目任务书

## 项目概述

为辽宁省某铁路职业技术学院开发智能实训能力评估平台 Demo。

**项目代号**: `liaoning-school-training`
**客户**: 辽宁某铁路职业技术学院
**目标**: 展示完整功能的 Demo，用于销售演示

## 核心功能模块

### 1. 数据同步
- 从校方 MySQL 数据库定时同步实训操作记录
- 每日自动同步，支持数据校验与去重

### 2. 实训成绩分析
- 根据逐步骤操作结果自动计算成绩
- 支持自定义评分规则
- 标注未通过步骤及对应能力要求

### 3. 学生能力模型与图谱
- 将实训表现映射至能力体系（大类能力 → 子能力）
- 生成能力雷达图（5-6 维度）
- 评估毕业标准达成度

### 4. 智能诊断报告（AI）
- 使用 LLM 生成个性化诊断报告
- 描述能力现状、薄弱环节、提升建议
- 支持单次实训报告和阶段性综合报告

### 5. 实训室环境规范检查（AI 图像）
- 上传实训室照片，AI 对比标准照片
- 检测器材归位、台面整洁、遗留杂物
- 结果纳入实训评价

### 6. 平台门户
- **学生端**: 查看成绩、能力图谱、诊断报告、下载
- **教师端**: 查看班级数据、统计汇总、导出
- **大屏展示**: 实训整体情况（展厅一楼大屏）

## 技术方案

### 前端
- React + Vite (TypeScript)
- 配色：**蓝色工业风**，类似铁路大屏信息平台
- 包含大屏展示模块（responsive，支持 1920x1080）

### 后端
- Python FastAPI
- SQLite（Demo 阶段，方便演示）
- AI 调用：通过 llm-proxy.densematrix.ai

### AI 集成
- **图像检测**: 使用 vision model（如 gemini-2.5-flash）
- **诊断报告生成**: 使用 LLM（如 gemini-2.5-flash）
