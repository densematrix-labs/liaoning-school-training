## 核心结论

### ✅ 技术可行性
- VLM 可以做实训视频审计，但不是"开箱即用"
- 预期准确率：70-75%（需要人工复核机制兜底）
- 每种实训场景需要独立的 Prompt 设计和调优

### ⚠️ 关键风险
1. **场景多样性**：每新增一种实训类型 = 额外的工程工作
2. **数据质量**：视频拍摄规范直接影响 AI 识别效果
3. **评分标准**：需要老师提供可量化的书面标准
4. **隐私合规**：是否能用云 API 决定技术路线

### 💰 20万能做什么
- 一个学校、一期、2-3种实训场景
- AI 辅助评分 + 人工复核
- 不是"完全自动化"或"100%准确"
