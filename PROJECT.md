## VLM 技术方案详解

### 为什么 VLM 可以做实训检测？

VLM（Vision-Language Model）是能同时理解图像和文字的 AI 模型。它可以：
- 看懂视频中的物体、人物、动作
- 理解给定的评分标准（文字描述）
- 按标准判断视频内容是否符合要求

### 工作流程

以**消防灭火器操作**为例：

```
┌──────────────────────────────────────────────────────────────┐
│                      输入                                      │
│  学生实训视频（30秒）+ 评分标准表                               │
└──────────────────────────────────────────────────────────────┘
                              ↓
┌──────────────────────────────────────────────────────────────┐
│                   Step 1: 视频预处理                           │
│  - 每秒抽取 1-2 帧关键画面                                     │
│  - 30 秒视频 → 30-60 张图片                                    │
│  - 图片质量增强（去模糊、调光）                                  │
└──────────────────────────────────────────────────────────────┘
                              ↓
┌──────────────────────────────────────────────────────────────┐
│                   Step 2: VLM 逐帧分析                         │
│  输入给 VLM：                                                   │
│  - 评分标准（5 个步骤，每步的正确/错误标准）                     │
│  - 示范视频截图（正确操作 vs 错误操作）                          │
│  - 当前帧图片                                                   │
│                                                                 │
│  VLM 输出：                                                     │
│  - 当前帧识别到的动作/状态                                       │
│  - 是否有安全隐患                                                │
│  - 关键物体检测（灭火器、保险销、火焰）                          │
└──────────────────────────────────────────────────────────────┘
                              ↓
┌──────────────────────────────────────────────────────────────┐
│                   Step 3: 时序整合                             │
│  - 把各帧的判断结果按时间排序                                    │
│  - 判断操作步骤是否按正确顺序完成                                │
│  - 标记每个步骤的开始/结束时间点                                 │
└──────────────────────────────────────────────────────────────┘
                              ↓
┌──────────────────────────────────────────────────────────────┐
│                   Step 4: 评分生成                             │
│  按评分标准逐项打分：                                            │
│  ✅ 取下灭火器（10/10）                                         │
│  ✅ 拔出保险销（20/20）                                         │
│  ⚠️ 握住喷管（15/20）- 扣分：握持位置偏后                       │
│  ❌ 对准火焰根部（10/25）- 扣分：对准了火苗而非根部              │
│  ✅ 压下把手喷射（25/25）                                       │
│  ──────────────────                                            │
│  总分：80/100                                                   │
└──────────────────────────────────────────────────────────────┘
                              ↓
┌──────────────────────────────────────────────────────────────┐
│                      输出                                      │
│  - 总分与等级                                                   │
│  - 每个步骤的得分与扣分原因                                      │
│  - 关键时间点截图标注（哪里做错了）                              │
│  - 改进建议                                                     │
└──────────────────────────────────────────────────────────────┘
```

### 评分标准示例（消防灭火器）

```yaml
实训项目: 灭火器使用
总分: 100

步骤:
  - 名称: 取下灭火器
    分值: 10
    正确标准: 一手握住把手，一手托底，稳定取下
    常见错误:
      - 单手拎起（-5分）
      - 倒置拿取（-10分）
    
  - 名称: 拔出保险销
    分值: 20
    正确标准: 用力拔出保险销，拔出前不能压把
    常见错误:
      - 没拔保险销就压把（-20分，无法喷射）
      - 拔出后保险销掉地（-2分，轻微）
    
  - 名称: 握住喷管
    分值: 20
    正确标准: 一手握喷管前端（距喷嘴 10-20cm）
    常见错误:
      - 握住喷嘴本身（-5分，有冻伤风险）
      - 握持位置过于靠后（-5分，控制不精准）
    
  - 名称: 对准火焰根部
    分值: 25
    正确标准: 喷嘴对准火焰底部（燃烧物表面）
    常见错误:
      - 对准火苗上部（-15分，无效灭火）
      - 距离火源过远（-10分，药剂浪费）
    
  - 名称: 压下把手喷射
    分值: 25
    正确标准: 均匀压把，左右扫射覆盖火焰
    常见错误:
      - 一直对准一点喷（-10分）
      - 喷射时间过短（-10分）
```

### VLM 能力边界

| 检测项 | VLM 能力 | 实现难度 | 解决方案 |
|--------|----------|----------|----------|
| **物体识别** | ✅ 强 | 低 | 通用 VLM 即可 |
| 灭火器、保险销、喷管 | | | |
| **基础动作** | ✅ 中等 | 中 | Prompt 设计 + 示例 |
| 拔、握、压、喷 | | | |
| **细节判断** | ⚠️ 需要训练 | 高 | Few-shot + 微调 |
| 握持位置、对准位置 | | | |
| **时序判断** | ⚠️ 需要设计 | 中 | 多帧分析 + 逻辑整合 |
| 步骤顺序是否正确 | | | |
| **安全隐患** | ✅ 中等 | 中 | 明确定义 + Prompt |
| 危险动作、错误操作 | | | |

### 需要校方配合提供

1. **评分标准文档**（必须）
   - 每个实训项目的标准评分表
   - 每个步骤的正确/错误判定标准
   - 扣分规则

2. **示范视频**（强烈建议）
   - 正确操作的标准视频
   - 常见错误的示例视频
   - 用于训练和测试 VLM 的识别能力

3. **历史评分数据**（如果有）
   - 人工评分的结果
   - 用于验证 AI 评分的准确性

### 技术风险与应对

| 风险 | 影响 | 应对措施 |
|------|------|----------|
| 视频质量差（模糊、光线暗） | 识别准确率下降 | 视频预处理增强 + 对拍摄提出最低要求 |
| 动作细节难以区分 | 细粒度打分困难 | 用示例视频做 few-shot，必要时微调模型 |
| 不同专业差异大 | 需要多套标准 | 初期聚焦 1-2 个专业，标准可配置 |
| VLM 产生幻觉 | 误判 | 多次推理取众数 + 人工复核机制 |
