🔬 VLM 能力边界

✅

强：物体识别

识别灭火器、工具、设备等常见物体，准确率高

✅

中等：基础动作

识别拔、握、压、拿等基础动作，需要 Prompt 优化

⚠️

弱：细粒度判断

判断握持位置精确到厘米、对准位置等细节，需要定制

⚠️

中等：时序推理

判断操作步骤顺序是否正确，需要多帧分析设计

⚠️ 关键数据： TemporalBench 基准测试显示，GPT-4o 细粒度时序理解准确率仅 38.5%，与人类（67.9%）存在约 30% 差距。

🚨 核心风险点

规模与性能

视频量级、并发需求、响应时间预期——100个 vs 10000个视频，架构完全不同

模型能力

API vs 本地部署的选择——数据出境合规问题，开源模型能力差距

场景多样性

每种实训类型需要独立的 Prompt 设计——没有一套方案通吃所有场景

数据质量

视频拍摄规范直接影响识别效果——垃圾视频 = AI 失效

评分标准

需要老师提供可量化的书面标准——"感觉"无法转化为代码

用户预期

管理期望值——AI 不是 100% 准确，需要人工复核机制
